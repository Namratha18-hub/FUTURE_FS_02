# 🚀 LeadFlow CRM – Client Lead Management System

A modern, full-featured **Client Lead Management System (Mini CRM)** designed to help businesses efficiently manage leads, track progress, and analyze performance using interactive dashboards and charts.

---

## 📌 Project Overview

LeadFlow CRM is a real-world business application that allows users to:

- Manage client leads
- Track lead status
- Maintain follow-ups
- Visualize data using charts & analytics

This project demonstrates full-stack application architecture using modern tools and best UI/UX practices.

---

## 🎯 Key Features

### 🔐 Authentication
- Secure Email & Password login/signup
- User-specific data (each user sees only their own leads)

---

### 📋 Lead Management
- Add new leads
- View leads in table format
- Edit lead details
- Delete leads with confirmation
- Lead status tracking:
  - New
  - Contacted
  - Converted

---

### 📝 Notes & Follow-ups
- Add multiple notes per lead
- Timestamp-based note history
- Organized follow-up tracking

---

### 📊 Dashboard & Analytics

The dashboard provides real-time insights into lead performance.

#### 🔹 Summary Cards
- Total Leads
- New Leads
- Contacted Leads
- Converted Leads
- Conversion Rate

---

#### 📈 Charts & Visualizations

- **Bar Chart** → Lead count by status  
- **Pie Chart** → Lead distribution percentage  
- **Line Chart** → Lead trends over time  
- **Conversion Indicator** → Success rate visualization  

---

### 🔍 Advanced Features
- Search leads by name/email
- Filter by status
- Sort leads
- Responsive design (mobile-friendly)
- Smooth animations and UI transitions

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS

**Backend**
- Lovable Cloud (Database, Authentication, APIs)

**Charts**
- Recharts / Chart.js
